WiFi Initialization Help:

SSID: Your WiFi Network Name
Password: Your WiFi Network Password (Leave Blank For No Password)

Default Gateway: Your Router's IP Address
Finding It: Run "ipconfig" in the command prompt. Commonly 192.168.1.1 or 192.168.0.1 or 10.0.0.1.

Static IP: Desired IP Address for the Smart Port Adapter
Picking One: Change just the last number of your default gateway to something else from 2-255.
Note: This is the address you'll type into the web browser on devices connected to your local network.

Subnet Mask: Your Network's Subnet Mask
Finding It: Run "ipconfig" in the command prompt. Commonly 255.255.255.0.
