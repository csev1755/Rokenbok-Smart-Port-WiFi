Help and Documentation


Vehicle Status:
• The color of the vehicle box corresponds to the controller selecting that key.
• Hover your mouse over a vehicle box to see the name of that vehicle.
• Vehicle names are changed by the system owner using the admin page.

Controller Status:
• Hover your mouse over a controller box to see the user name of the person using that controller.
• You can change your username by clicking on your controller name ("Keyboard" or "Controller X").

System Updates:
• Hold down the reset button on the device until the system goes back into initialization mode.
• Connect to the "Smart-Port-Provisioning" network and navigate to 1.2.3.4 in a browser.
• Upload the .bin file to the update menu to commence the upgrade.

Keyboard Controls:
Enable/Disable Keyboard = 9/0
Decrease/Increase Selection = 1/2
UDLR = W/A/S/D
ABXY = F/R/Q/E
Slow = Shift
